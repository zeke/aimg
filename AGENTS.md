# AGENTS.md

- This is a Node.js ESM CLI for generating images with Replicate.
- Entry point: `index.js`. Image generation and download logic lives in `lib/run.js`.
- Run `npm test` before claiming changes are done. This runs StandardJS linting.
- Run `npm run lint` to auto-fix StandardJS formatting issues.
- Keep CLI docs in `README.md` in sync with defaults in `index.js`.
- Revise this file whenever meaningful project-specific workflow, scripts, conventions, or gotchas change.
